Workout.cool

Una plataforma moderna de entrenamiento fitness

Sobre el Proyecto

Workout.cool es una plataforma integral de entrenamiento fitness que permite a los usuarios crear planes de entrenamiento, realizar seguimiento de su progreso y explorar una amplia base de datos de ejercicios.

Características Principales

Planes de Entrenamiento

Crea rutinas de ejercicio personalizadas.

Seguimiento de Progreso

Monitorea ejercicios y sesiones de entrenamiento.

Base de Datos de Ejercicios

Accede a ejercicios con demostraciones en video.

Soporte Multilenguaje

Soporte interactivo para usuarios internacionales.

Tecnologías Utilizadas

  • Next.js / React
  • PostgreSQL / Prisma
  • Better Auth
  • Tailwind CSS
  • TanStack Query
  • Zustand
